How to use

  1. Add content with the form.
  2. Click Save JSON to save `family-content.json`.
  3. Click Update Website and choose `app.js` when asked.
  4. Open `index.html` to see the updated family website.

If your browser does not support direct file saving, the manager will download the JSON file instead.
